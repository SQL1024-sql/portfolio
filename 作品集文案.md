## 01 ｜晚晴餐廳 — 高級餐廳形象官網

**類型**：餐飲 ｜**技術**：HTML / CSS、響應式設計

## 02 ｜晨光咖啡 — 一頁式品牌頁
為獨立咖啡館設計的一頁式（Landing Page）品牌網站。以溫暖色調與簡潔排版突顯品牌個性，包含特色介紹、人氣菜單與門市資訊，適合小型店家快速建立網路門面。
**類型**：餐飲 ｜**技術**：HTML / CSS、一頁式設計

## 03 ｜PULSE 健身房 — 會員方案介紹

**類型**：運動健身 ｜**技術**：HTML / CSS、方案比較版型

## 04 ｜微笑牙醫診所 — 診所官網

**類型**：醫療診所 ｜**技術**：HTML / CSS、響應式設計

**類型**：專業服務 ｜**技術**：HTML / CSS、網格版型

## 06 ｜FRAME 攝影工作室 — 作品展示網站

**類型**：創意 / 攝影 ｜**技術**：HTML / CSS、瀑布流相簿

## 07 ｜森林之丘 — 預售屋建案銷售頁

**類型**：不動產 ｜**技術**：HTML / CSS、行銷型 Landing Page

**類型**：個人品牌 ｜**技術**：HTML / CSS、響應式設計

**類型**：電商 ｜**技術**：HTML / CSS / **JavaScript**、購物車互動
